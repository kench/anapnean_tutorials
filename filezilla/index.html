Using FileZilla
FileZilla is a moderatly popular SFTP client. Anapnea does not have a FTP server, because FTP is insecure. Instead you can use SFTP, which isn't strictly FTP over SSH, but it's close. FileZilla uploads and downloads files from Anapnea only; if you want to use the terminal, you'll need a normal SSH client. (Windows, Linux or Mac).
Start by installing FileZilla. If you're on Mac or Windows, you probably want to download it from their website. Linux users should probably use their built-in package management system. Below, I'm using Synaptic on Ubuntu as an example.
The FileZilla wiki has a fairly comprensive Tutorial, so I'm only going to cover the basics. There's tonnes of documentation at that link.
Firstly, open FileZilla, and type sftp://anapnea.net into the Host box, and put your username and password in their respective boxes. Press QuickConnect to connect to Anapnea and verify the fingerprint. The advice that can be found at the bottom of the Connecting to SSH article with regards to fingerprints remains important and still applies here.
The left hand side represnts files on your computer, and the right hand side represents files on Anapnea. You can navigate both the same way you might do in Nautilus (Linux) or Explorer (Windows). Files are copied from one to the other by dragging and dropping.
The transfer should appear in the Succesful transfers tab to let you know that it completed. The file you coped will now appear in its new location.
SFTP operates on the same port as SSH, that is, by default, 22. If you need to use port 443, then type sftp://anapnea.net:443 into the Host box instead of just sftp://anapnea.net.
